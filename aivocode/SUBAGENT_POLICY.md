# Subagent Policy

- **`@explore`**: Use for codebase discovery/exploration, symbol/file lookup, multi-file search, and
  targeted snippet extraction. Ask for file paths, minimal relevant snippets, and short
  explanations; avoid full files.

- **`@web-ops`**: Use for all web/external documentation lookup, search, and fetch tasks.
  Ask for exact facts, concise summaries, and source URLs.

- **`@general`**: Use for synthesis, organization, or well-scoped general tasks.
